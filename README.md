# 如何在本地環境執行專案

## 環境變數

| 環境變數  | description |
|----------------|-----------------------|
| BOT_TOKEN      |   Discord bot token   |
| CHATGPT_TOKEN  |   OpenAI API key      |
| DEPLOYMENT_ENV |   [beta, prod]        |

如何設定環境變數
export {環境變數}=..

範例

```
export DEPLOYMENT_ENV=beta
```

## 執行

打包 jar

```
> ./mvnw clean package
```

執行

```sh
java -jar main/target/main-1.0-SNAPSHOT-jar-with-dependencies.jar
```
